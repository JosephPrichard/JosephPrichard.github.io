My Projects

3DGR

3D Graphical Renderer is a Web Application that runs on Glassfish, uses back-end technologies Java EE, Jsps/Jstl, and SQL, and front end technologies Html, CSS, Javascript, and WebGl. It provides interactive 3D rendering. Projects can be created with a variety of primitive and compound shapes, written through a comfortable UI or custom scripts to speed up project creation. Projects can be viewed on a 3D Canvas powered by WebGL that allows for rotating, zooming in/out, and camera translations! Sign up to start rendering, and post your renderings publicly when you are done! I take web security very seriously, all passwords are hashed, anti-XSS and anti-Sql Injection measures have been taken. Watch a Video demonstration, or see the source code on my Github.

Funnel Finder

Funnel Finder is a Desktop Application powered by Java and Java API. It provides a GUI to create funnel translations between shapes (N-gons and squares). Funnel translations are outputted as CAD scripts so they can be viewed in 3D. Previous projects are saved on a local database so you can keep track of funnel scripts you generate. The image on the right is a circle to square translation rendered with my other project, 3DGR. This project's source code can be viewed on my Github.
